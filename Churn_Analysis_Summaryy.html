What is Churn Analysis?

📌 About the Project

This project performs an end-to-end churn analysis for a telecom firm using advanced data analytics and machine learning to predict which customers are at risk of leaving.

🎯 Project Goals

  • Visualize & analyse customer data at demographic level
  • Analyse geographic distribution of churn
  • Study payment & account information patterns
  • Identify churner profile for marketing campaigns
  • Predict future churners using ML

📊 Metrics Tracked

  • Total Customers — 6,418
  • Total Churn — 1,732
  • Churn Rate — 26.99%
  • New Joiners — 411
  • Predicted Future Churners — 384

🏭 Target Audience

Applicable across various industries — from retail and finance to healthcare. Any business that values customer retention can benefit from this approach.
